# Modern Philosophy

Notes on early modern philosophy, roughly from the 17th to the 19th century.

## Topics

- René Descartes (rationalism, mind-body problem, cogito)
- Baruch Spinoza
- Gottfried Wilhelm Leibniz
- John Locke (empiricism, political philosophy)
- George Berkeley (idealism)
- David Hume (skepticism, causality, empiricism)
- Immanuel Kant (Critique of Pure Reason, deontological ethics)
- G.W.F. Hegel (dialectics, idealism)
